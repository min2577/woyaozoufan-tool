# docs

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放文档的文件夹，主要用来存放服务器相关的文档。简单来说，就是项目的各种文档，比如 API 文档等，都存放在这里。

## 2. 里面存放哪些内容？
- **openapi.yaml**：API 文档文件，使用 OpenAPI 规范定义的 API 接口文档

## 3. 哪些文件绝对不能删？
- openapi.yaml：API 文档文件，删了就没有 API 接口的文档了

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里只有一个 openapi.yaml 文件，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
不影响。这个文件夹里的文件只是文档，不影响服务器的正常运行。

## 6. 我可以在这里做什么？
- 查看 API 文档
- 修改 API 文档
- 添加新的文档文件

## 7. 注意事项（非常重要）
- 不要删除 openapi.yaml 文件，否则就没有 API 接口的文档了
- 修改 API 文档时，要确保与实际的 API 接口一致
- 文档要保持更新，反映最新的 API 接口情况
