# agents

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放代理脚本的文件夹，主要用来存放与代理相关的脚本文件。简单来说，就是一些用于代理功能的脚本，比如代理指挥官、代理契约守护者、代理提示优化器等，都存放在这里。

## 2. 里面存放哪些内容？
- **agent_commander.js**：代理指挥官脚本，可能用于管理和指挥其他代理
- **agent_contract_guardian.js**：代理契约守护者脚本，可能用于保护和管理契约
- **agent_prompt_optimizer.js**：代理提示优化器脚本，可能用于优化提示信息

## 3. 哪些文件绝对不能删？
这些脚本文件都可以删除，因为它们只是辅助工具，不会影响服务器的正常运行。但如果你需要使用这些代理功能，就不要删除。

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里的文件都是代理脚本，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
不影响。这个文件夹里的文件只是辅助脚本，不影响服务器的正常运行。

## 6. 我可以在这里做什么？
- 修改现有的代理脚本
- 添加新的代理脚本
- 调整代理脚本的逻辑

## 7. 注意事项（非常重要）
- 运行脚本时要注意，有些脚本可能会影响代理功能
- 要了解脚本的功能，避免误操作
- 脚本的命名要规范，便于理解和维护
