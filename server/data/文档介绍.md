# data

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放数据文件的文件夹，主要用来存放服务器运行时需要的数据。简单来说，就是项目中使用的各种数据，比如菜谱数据、营养数据、数据库文件等，都存放在这里。

## 2. 里面存放哪些内容？
- **recipe_repair_reports/**：菜谱修复报告文件夹，包含了菜谱修复的报告文件
- **dish_names.json**：菜名数据文件
- **inventory_report.json**：库存报告文件
- **migrate_to_sqlite_test.db**：SQLite 迁移测试数据库文件
- **nutrition-db.json**：营养数据库文件
- **recipes.db**：菜谱数据库文件
- **recipes.db-shm**：菜谱数据库的共享内存文件
- **recipes.db-wal**：菜谱数据库的预写日志文件
- **recipes.json**：菜谱数据文件
- **simple_dish_names.json**：简单菜名数据文件
- **woyaozoufan.db**：我要揍饭数据库文件
- **菜谱备份_422道.zip**：菜谱备份文件

## 3. 哪些文件绝对不能删？
- **recipes.db**：菜谱数据库文件，删了菜谱数据就没了
- **woyaozoufan.db**：我要揍饭数据库文件，删了应用数据就没了
- **nutrition-db.json**：营养数据库文件，删了营养数据就没了
- **recipes.json**：菜谱数据文件，删了菜谱数据就没了

## 4. 哪些是备份/旧版本/无用文件？
- **migrate_to_sqlite_test.db**：测试数据库文件，可能是迁移时的测试文件
- **菜谱备份_422道.zip**：备份文件，是菜谱的备份
- **recipe_repair_reports/**：修复报告文件夹，可能是临时生成的报告

## 5. 这个文件夹是否影响项目正常运行？
是的，非常影响。如果这个文件夹不存在或者数据文件有问题，服务器就无法正常运行，因为缺少必要的数据。

## 6. 我可以在这里做什么？
- 查看数据文件的内容
- 备份数据文件
- 导入新的数据文件
- 清理不需要的数据文件

## 7. 注意事项（非常重要）
- 不要删除核心数据文件，否则数据会丢失
- 数据库文件（.db 文件）不要随意修改，否则可能会损坏
- 备份重要的数据文件，防止数据丢失
- 数据文件可能会比较大，注意存储空间的使用
