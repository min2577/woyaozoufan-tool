# utils

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放工具函数的文件夹，主要用来存放服务器的各种工具函数。简单来说，就是一些可以在多个地方复用的代码，比如食材解析、Markdown 转 JSON、菜谱契约、词汇表等功能的实现。

## 2. 里面存放哪些内容？
- **ingredientParser.js**：食材解析工具函数，用于解析食材信息
- **markdownToJsonEngine.js**：Markdown 转 JSON 引擎，用于将 Markdown 转换为 JSON 格式
- **recipeContractV0005.js**：菜谱契约版本 0005，用于定义菜谱的数据结构和验证规则
- **ssotLexicon.js**：单一真实来源词汇表，用于定义系统中使用的词汇

## 3. 哪些文件绝对不能删？
- **ingredientParser.js**：食材解析工具函数，删了就无法解析食材信息
- **markdownToJsonEngine.js**：Markdown 转 JSON 引擎，删了就无法将 Markdown 转换为 JSON
- **recipeContractV0005.js**：菜谱契约，删了就无法验证和处理菜谱数据
- **ssotLexicon.js**：词汇表，删了就无法使用系统中定义的词汇

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里的文件都是工具函数，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
是的，非常影响。如果这个文件夹不存在或者工具函数文件有问题，服务器就无法正常处理相应的功能。

## 6. 我可以在这里做什么？
- 修改现有的工具函数
- 添加新的工具函数
- 调整工具函数的逻辑

## 7. 注意事项（非常重要）
- 不要删除工具函数文件，否则对应的功能无法使用
- 修改工具函数时，要确保功能正常
- 工具函数的命名要规范，便于理解和维护
- 要注意工具函数的性能，避免影响服务器的运行速度
