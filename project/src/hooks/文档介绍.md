# hooks

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放自定义钩子的文件夹，主要用来存放 React 项目的各种自定义钩子。简单来说，就是一些可以复用的逻辑代码，通过钩子的形式提供给组件使用。

## 2. 里面存放哪些内容？
- **index.ts**：钩子的导出文件
- **useDebounce.ts**：防抖钩子，用于处理防抖逻辑
- **useHotkeys.ts**：热键钩子，用于处理键盘快捷键
- **useInfiniteScroll.ts**：无限滚动钩子，用于处理无限滚动逻辑
- **useLocalStorage.ts**：本地存储钩子，用于处理本地存储
- **useToast.ts**：提示消息钩子，用于显示提示消息

## 3. 哪些文件绝对不能删？
- 所有的钩子文件都不能删，否则对应的逻辑复用功能就无法使用
- index.ts：钩子的导出文件，删了就无法导出钩子
- useDebounce.ts：防抖钩子，删了就无法使用防抖功能
- useHotkeys.ts：热键钩子，删了就无法使用键盘快捷键
- useInfiniteScroll.ts：无限滚动钩子，删了就无法使用无限滚动功能
- useLocalStorage.ts：本地存储钩子，删了就无法使用本地存储功能
- useToast.ts：提示消息钩子，删了就无法显示提示消息

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里的文件都是自定义钩子，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
是的，非常影响。如果这个文件夹不存在或者钩子文件有问题，项目中使用这些钩子的功能就无法正常工作。

## 6. 我可以在这里做什么？
- 修改现有的钩子
- 添加新的钩子
- 调整钩子的逻辑

## 7. 注意事项（非常重要）
- 不要删除钩子文件，否则对应的逻辑复用功能无法使用
- 修改钩子时，要确保功能正常
- 钩子的命名要规范，便于理解和维护
- 这个项目使用了 TypeScript，所以要注意类型定义
