# .bolt

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放 Bolt 相关配置的文件夹，Bolt 可能是一个工具或框架。简单来说，就是用来配置 Bolt 工具的相关设置和提示信息的文件夹。

## 2. 里面存放哪些内容？
- **config.json**：Bolt 的配置文件，包含了 Bolt 工具的各种配置参数
- **prompt**：可能是 Bolt 工具使用的提示信息文件

## 3. 哪些文件绝对不能删？
如果你在使用 Bolt 工具，那么这些文件不能删。否则，Bolt 工具可能无法正常工作。

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里的文件都是 Bolt 相关的配置文件，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
如果项目使用了 Bolt 工具，那么这个文件夹会影响项目的正常运行。如果没有使用，那么不影响。

## 6. 我可以在这里做什么？
- 修改 Bolt 的配置参数
- 更新提示信息

## 7. 注意事项（非常重要）
- 不要删除这个文件夹里的文件，否则 Bolt 工具可能无法正常工作
- 修改配置文件时，要确保配置格式正确
- 要了解 Bolt 工具的使用方法，才能正确配置
