# public

## 1. 这个文件夹是干嘛的？（大白话）
这是一个存放静态资源的文件夹，里面的文件会被直接复制到构建后的 dist 文件夹中，不会被编译。简单来说，就是存放一些不需要编译的静态文件，比如 PWA 相关的配置文件、离线页面等。

## 2. 里面存放哪些内容？
- **manifest.json**：PWA（渐进式 Web 应用）的配置文件，包含了应用的名称、图标、主题色等信息
- **offline.html**：离线页面，当用户没有网络时显示
- **sw.js**：Service Worker 文件，用于实现 PWA 功能，比如离线缓存、推送通知等

## 3. 哪些文件绝对不能删？
如果你需要使用 PWA 功能，那么这些文件不能删。否则，PWA 功能会无法正常工作。

## 4. 哪些是备份/旧版本/无用文件？
这个文件夹里的文件都是 PWA 相关的配置文件，没有备份或旧版本文件。

## 5. 这个文件夹是否影响项目正常运行？
如果项目需要使用 PWA 功能，那么这个文件夹会影响项目的正常运行。如果不需要，那么不影响。

## 6. 我可以在这里做什么？
- 修改 PWA 的配置信息
- 更新离线页面的内容
- 修改 Service Worker 的逻辑
- 添加新的静态资源文件

## 7. 注意事项（非常重要）
- 这个文件夹里的文件会被直接复制到 dist 文件夹中，不会被编译
- 如果你需要引用这个文件夹里的文件，应该使用绝对路径，比如 /manifest.json
- 不要在这个文件夹里存放需要编译的文件，比如 JavaScript 或 CSS 文件
- 修改 PWA 相关文件时，要确保配置正确，否则 PWA 功能可能无法正常工作
